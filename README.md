To start the project, you need to write the npm run start command in the terminal, and since I have attached the concurrently package, the backend and the frontend will be opened together. In this project, it is possible to select your desired candidate, go to his separate page, write comments, read his biography. The backend for the project is the json server. I used framer motion to get nice animation.
